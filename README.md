# track-trash

Installation:
npm init -y
npm install express mysql2 dotenv bcryptjs jsonwebtoken cors


## Setup

1. Clone the repository

git clone <repository-url>

2. Install backend dependencies

npm install

3. Start backend

npm run dev

Backend runs on: http://localhost:5001

4. Start frontend

cd frontend  
npm install  
npm run dev  

Backend runs on: https://track-trash.onrender.com

5. Expose the frontend

ngrok http 5173

Use the generated ngrok URL to access the application externally
